# بيانات منصة سَنَد

هذا المستودع يحوي **البيانات المنشورة فقط** لمنصة سَنَد — ملفات JSON والوسائط التي تعرضها الصفحة.

🔗 المنصة: https://sanad-sovereign-news-intelligence.netlify.app

| الملف | المحتوى |
|---|---|
| `daily/news.json` | الأخبار المُسنَدة بأقسامها |
| `daily/intel.json` | حصيلة الأزمة بمصادرها |
| `daily/verify.json` | نتائج المطابقة المستقلة |
| `daily/official.json` | البيانات الرسمية |
| `daily/latest.json` | نشرة اليوم |
| `daily/archive/` | لقطات الجولات السابقة |

البيانات متاحة للاستخدام مع ذكر المصدر.
