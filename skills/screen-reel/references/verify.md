# فحوص لا تُتخطّى

كل فحص هنا وُلد من عطلٍ حقيقيّ نجا من المراجعة بالنظر.

## ١ · تغطية الإطار

الصفحة يجب أن تملأ ≥٨٥٪ من عرض الإطار. أقلّ من ذلك يعني أن `viewport` و`recordVideo.size` غير متطابقين.

```python
from PIL import Image
import numpy as np
a = np.array(Image.open("frame.png").convert("L")); H,W = a.shape
cols = (a>25).sum(axis=0)
nz = [i for i,v in enumerate(cols) if v > H*0.05]
print(f"{round((nz[-1]-nz[0])/W*100)}% عرضًا")
```

## ٢ · المنطقة الآمنة السفلية

واجهة إنستغرام تغطّي أسفل الإطار. المقصود ألّا يقع **العنوان** هناك — أمّا محتوى الصفحة المسجَّلة فيمتدّ للأسفل طبيعيًّا ولا يُعدّ خطأً.

قِس شريط العنوان وحده:

```python
cap = a[H-470:H-170,:]           # نطاق العنوان
below = a[H-170:,:]              # ما تحته
print((cap>150).sum(), (below>150).sum())
# المطلوب: العنوان > 1500، وألّا يكون ما تحته عنوانًا (قارِن بصريًّا)
```

## ٣ · ظهور العنوان

```python
band = a[H-470:H-170,:]
print((band>150).sum())           # المطلوب > 1500
```

## ٤ · النطق

`verify_voice.py` يفرّغ الصوت ويقارن كلمةً كلمة بعد تجريد الحركات. أخطاء ظهرت فعليًّا:

| المكتوب | المسموع | السبب | الحل |
|---|---|---|---|
| `يشتغل` | يشترل | الغين تُلبس الراء | استبدل بـ`يعمل` |
| `بَنَيْتُ` | بُنيب | الفعل بلا سياق | أضف `أنا` قبله |
| `خبرٍ` | خِبْرة | تنوين بلا فتحة | `خَبَرٍ` — تشكيل كامل |

## ٥ · الإيقاع

١٢٠–١٥٠ كلمة/دقيقة. هذا المقياس وحده استبعد نموذجًا بديلًا كان أنظف صوتًا لكنه تكلّم بـ٥٨ ك/د.

قِسها على **الكلام وحده** لا على الملفّ الممدود — الصمت المضاف عند التركيب يجعل الإيقاع يبدو بطيئًا كذبًا (٣٦ ك/د بدل ١٢٢).

```python
import librosa
y,sr = librosa.load(path, sr=22050)
rms = librosa.feature.rms(y=y, frame_length=1024, hop_length=256)[0]
speech = (rms > max(rms.max()*0.06, 0.008)).sum() * 256 / sr
print(len(text.split()) / speech * 60)
```
