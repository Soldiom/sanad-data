# إخراج المشهد

التمرير وحده ممل. المشهد الجيّد يقود العين.

## البنية

```
① هويّة    ١٫٥–٢ث   الشعار أو الترويسة تستقرّ
② سياق     ٤–٦ث     تمرير لمنطقة المحتوى
③ تركيز    ٢–٣ث     عنصر واحد يُضاء
④ تفصيل    ٢–٣ث     تكبير تدريجيّ للتفصيلة الحاسمة
⑤ إثبات    ٣–٤ث     رقم أو شارة أو مصدر
⑥ ذروة     ٣–٤ث     أكثر عنصر حركةً في الصفحة
```

## ملف الإيقاعات

```json
[
  {"scroll":"#feed .item","pauses":2},
  {"glow":"#feed .item"},
```

```bash
node scripts/record.js https://example.com /tmp/vid beats.json
```

## قواعد

- **سجّل ٦٠–٩٠ ثانية واقتطع ٢٠**. المونتاج يصنع الإيقاع لا التصوير.
- **لا قفزة تكبير واحدة** — ثلاث مراحل تُقرأ كتركيزٍ لا كخلل.
- **الإضاءة قبل التكبير** بثمانمئة مللي ثانية على الأقل، وإلا فات المشاهدَ ما تُشير إليه.
- **الذروة في النهاية** — أكثر ما في الصفحة حركةً يُحفظ للأخير.
